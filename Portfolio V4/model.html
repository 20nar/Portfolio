Backdrop

When a band performs on a live stage there is usually a backdrop. This is something behind the performance to enhance the overall experience of the show. This must compliment the performance and should not be all too distracting. I wanted to design a backdrop with the characters that we made in Sprint 2.


Pattern with colors

The colors we chose for the Nachtwakers really contrast with one another. I wanted to create this nice fluid motion that at one point shows the characters and their logo.


Animation

I grabbed that same image and imported them into Adobe Premiere Pro and worked with keyframes. By duplicating the image and letting them come into the frame with keyframes. It looks as if the colors are overlapping each other.


Animation with logo

I grabbed the same animation and imported the logo from the Nachtwakers. The only annoying thing about this logo is that the moon on the right pushes the font the left and therefore it looks like it is off center.


Backdrop Overlay

I removed the logo with the font and put an image of a live performance on top of it. By removing each section, the animation is revealed.


Backdrop Overlay

In the end I grabbed a live performance of the Nachtwakers and Put the same animation as before but then added the logo and one character of the 3D models that we made.


Conclusion

In the end I learned how to work with keyframes and discovered that backdrop is an important element to fesitvals and concerts these days. I would love to make more of these in the future. It can be quite a creative challenge, but backdrops are definitely necessary for bands performance.
