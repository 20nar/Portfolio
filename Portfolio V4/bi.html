Brandguide Inspiration

Logo

When we decided to make a brandguide for the band, we first started with some inspiration. I searched up a few band logos that might fit with the Nachtwakers. The Nachtwakers told us that we didn't necessarily have to change the logo. But I really wanted to see if we could get a more uniform logo for the Nachtwakers. Because according to our Target Group Research, nobody expected the Nachtwakers to play ska/reggae when they first saw that logo.


Logo structure

In order to design a logo for a brand, there has to be a structure and certain rules. So the logo gets used properly. Over here you can see how that goes.


Typography

This is also a very important aspect of the band. The typography will have certain rules. Headers and titles should use different fonts and of course the main text should have a different font as well. But they have to compliment one another. This will provide a pleasant reading experience for the reader. These rules will be applied to the website.
